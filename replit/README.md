# FlowTTS on Replit

基于腾讯云 FlowTTS 的语音合成演示 - Replit 版本

## 使用方法

1. Fork 此项目到你的 Replit 账号
2. 点击 Run 运行
3. 填写腾讯云凭证后即可使用

## 获取凭证

- [腾讯云 API 密钥](https://console.cloud.tencent.com/cam/capi) - 获取 SecretId 和 SecretKey
- [TRTC 控制台](https://console.cloud.tencent.com/trtc) - 获取 SdkAppId
